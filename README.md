version 0.13.1 explicitly via the Rust cargo package management system.

of several Rust versions.

When CV changes are made, these should be committed and pushed as one would in any other
repository.

To generate a new version of the published CV, a tag must be created, e.g.

```
git tag v1.0
git push --tags
```

The newly published version can be linked to at
`https://github.com/thombruce/cv/releases/latest/download/Thom-Bruce_CV.pdf`,
which should always point to the latest release.
